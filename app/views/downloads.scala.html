Downloads

If you want to run NavProt on your computer, you can download the zip archive below. Archive contains three files, navprot.jar which is the program itself, navprot which is the bash script for Linux/OS X and navprot.bat which is a command line script for running NavProt in Windows.


Files Size
navprot.zip 4.26 MB

Requirements

You need to have Java Runtime Environment (at least version 1.7) installed on your computer. To check it on Windows, you need to open command prompt. Press Windows Key + R and type cmd and press enter. For checking it on Linux/OS X open a terminal window. Type the following into console: 

java -version

If you see an output similar to this one, you are good to go:

java version "1.7.0_45"
Java(TM) SE Runtime Environment (build 1.7.0_45-b18)
Java HotSpot(TM) 64-Bit Server VM (build 24.45-b08, mixed mode)

If you do not have Java installed you can download it from here. For OS X you might have to install Java Development Kit as well, which can be found in here.

Usage

Linux/OS X

Run the terminal and change directory to directory where you extracted the contents of the zip archive. 

$ cd Downloads

Unzip the contents of the zip file. 

$ unzip navprot.zip

We are interested in navprot shell script. We need to make it executable. 

$ chmod u+x navprot

Let's assume that your PDB files are located in /home/user/mypdbs and initial.pdb is the name of your initial conformation PDB file and final.pdb is the name of your final conformation PDB file. For this example we take A chains of both structures. In addition, we want to save our results into /home/user/navprot_results. Type the following to start running your simulation. 

$ ./navprot /home/user/mypdbs/initial.pdb A /home/user/mypdbs/final.pdb A /home/user/navprot_results
# or
$ sh navprot /home/user/mypdbs/initial.pdb A /home/user/mypdbs/final.pdb A /home/user/navprot_results
Windows

Extract the contents of the zip archive that you downloaded. Press Windows Key + R and type cmd and press enter. Change directory to the directory where you extracted your files in the command prompt. 

> cd C:/Users/User/Downloads

navprot.bat is our executable file for running the simulation. Let's assume that we want to save our results to C:/Users/User/Documents/Navprot\ Results. If our PDB files are inside the Documents directory and initial.pdb and final.pdb are the names of initial and final conformation PDB files (A chains are considered for both), type the following to run the simulation: 

> navprot.bat C:/Users/User/Documents/initial.pdb A C:/Users/User/Documents/final.pdb A C:/Users/User/Documents/Navprot\ Results

Alternatively, for both Linux/OS X and Windows you can also use the following command: 

java -jar navprot.jar [path to initial conformation PDB] [initial PDB chain ID] [path to final conformation PDB] [final PDB chain ID] [path to results directory]

Simulation might take from a few hours up to a three or four days. When it is over, there will be three files in your results directory. torsion.txt contains the theta and phi angles of both initial and final structure. It is created in the beginning of the simulation and it is read by the program itself. results.txt file includes cartesian and generalized coordinates of each simulation step. Finally, results.pdb is created at the end of the simulation. It contains each conformation of each step taken during the simulation as NMR model.
